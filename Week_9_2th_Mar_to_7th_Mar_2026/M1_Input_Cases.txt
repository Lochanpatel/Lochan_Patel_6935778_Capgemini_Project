1. AUTHENTICATION STATE
----------------------------------------------------------------
HOW INPUT IS READ:
  Line 1       : Number of allowed locations
  Next N lines : Location names (one per line)
  Next line    : Number of users
  Next M lines : id,email,password,location   (comma-separated)
  Next line    : Number of actions
  Next P lines : Action:UserIndex  (e.g. Register:0, Login:1, Logout:0)

  India
  USA

  2,bob@mail.com,pass456,USA

  Register:2

  Login:1
  Login:0
  Logout:0

  alice@mail.com registered successfully!

  alice@mail.com logged in successfully!

8. GROCERY RECEIPT
----------------------------------------------------------------
NOTE: This program has HARDCODED data — no Console input is needed.
      The prices, discounts, and shopping list are fixed in code:

HARDCODED DATA:
  Prices   : Banana=10, Apple=20, Orange=8
  Discounts: Orange gets 10% off
  Shopping : Banana x2, Apple x3, Orange x5

EXPECTED OUTPUT (just run the program):
  Banana 10 20
  Apple 20 60
  Orange 8 36

  Calculation:
    Banana : 10 * 2 = 20  (no discount)
    Apple  : 20 * 3 = 60  (no discount)
    Orange : 8  * 5 = 40, then 10% off → 40 - 4 = 36

NO INPUT REQUIRED — just press Run/Enter.

14. USER MANAGEMENT
----------------------------------------------------------------
HOW INPUT IS READ:
  Line 1         : Number of existing DB users N
  Next N lines   : CSV row with 18 fields:
                   id,identityNumber,firstName,lastName,age,birthDate,
                   email,gender,country,city,address,zipCode,phone,
                   department,roles,joinDate,credit,status
                   (indexes used: 0=id, 2=firstName, 3=lastName, 6=email, 17=status)

  Next line      : Number of new users M
  Next M lines   : Same 18-field CSV format

  Logic:
    - If new user's Id == 0  → Inserted (new user)
    - If Id exists in DB and any of (firstName, lastName, email, status) changed → Updated

SAMPLE INPUT:
  3
  1,ID001,Alice,Smith,28,1996-01-01,alice@mail.com,Female,India,Delhi,Addr1,110001,9876543210,IT,Admin,2023-01-01,500.00,Active
